Effective Runs Calculator

Convert a set of runs into one decimal run count. More overlap between consecutive runs lowers the number, so better runs means a lower run count!

Formula

effectiveRuns = n - 1 100 ∑ i = 1 n - 1 ( b i - a i + 1 )

For each neighboring pair, subtract the next run's start from the previous run's best endpoint. Add those overlaps, divide by 100, then subtract from the integer run count.
